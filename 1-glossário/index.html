Glossário de alguns termos da internet

HTML
HiperText Markup Language, traduzindo ao português: Linguagem de Marcação de Hipertexto. O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Wikipédia
CSS
Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web.Google
WEB
A World Wide Web designa um sistema de documentos em hipermídia que são interligados e executados na Internet. Os documentos podem estar na forma de vídeos, sons, hipertextos e imagens. Wikipédia
XML
é uma recomendação da W3C para gerar linguagens de marcação para necessidades especiais. Wikipédia
SVG
Trata-se de uma linguagem XML para descrever de forma vetorial desenhos e gráficos bidimensionais, quer de forma estática, quer dinâmica ou animada.
